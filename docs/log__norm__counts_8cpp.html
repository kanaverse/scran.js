Compute log-normalized expression values.

Parameters
matA NumericMatrix object containing features in rows and cells in columns.
use_size_factorsWhether to use custom size factors, or to compute them based on the library size.
size_factorsOffset to an array of doubles of length nsubsets, containing (possibly uncentered) positive size factors for each column of mat. Only used if use_size_factors = true.
use_blocksWhether or not to compute the default filters within each block.
blocksOffset to an array of int32_ts with mat.ncol() elements, containing the block assignment for each cell. Block IDs should be consecutive and 0-based. If use_blocks = false, this value is ignored.
Returns
A NumericMatrix containing log-normalized expression values is returned. If use_blocks = true, centering of the size factors is performed within each block, so as to mimic separate normalization of cells in that block.
