Compute group-based size factors to remove composition biases in sparse data.

Parameters
matA NumericMatrix object containing features in rows and cells in columns.
[in]groupsOffset to an integer array of length equal to the number of columns in mat, specifying the group assignment for each cell. Group IDs should be consecutive and 0-based. If false, an average pseudo-cell is constructed from mat and used as the reference.
centerWhether to center the size factors so that the average is set to unity. If false, the size factors can be interpreted as the scaling to ref.
prior_countPrior count to use for shrinking size factors towards the relative library size. Larger values result in stronger shrinkage when the coverage is low.
{?number}reference Group to use as a reference. This should be an entry in the array pointed to by groups.
[out]outputOffset to a double-precision array of length equal to the number of cells in mat, to store the size factor for each cell.
Returns
output is filled with the size factors for all cells in mat.
