Parameters

runtimeNumber of milliseconds to run before returning. Iterations are performed until the specified runtime is exceeded. If runtime is not positive, it is ignored and the algorithm runs to maxiter.
maxiterMaximum number of iterations to perform. The function will return even if runtime has not been exceeded.
[in,out]YOffset to a two-dimensional array containing the initial coordinates. Each row corresponds to a dimension, each column corresponds to a cell, and the matrix is in column-major format. On output, this will be filled with the updated coordinates.
Returns
Y and TsneStatus are updated with the latest results.
