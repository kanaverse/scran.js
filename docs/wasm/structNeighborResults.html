Manually reconstruct the nearest neighbor search results, usually from a separate memory space.

Parameters
nNumber of observations.
[in]runsOffset to an integer array of length equal to the number of observations, containing the number of neighbors for each observation.
[in]indicesOffset to an integer array of length equal to the number of neighbors across all observations (i.e., size()). This contains the indices of the neighbors for each observation.
[in]distancesOffset to a double-precision array of length equal to the number of neighbors across all observations. This contains the distances to the neighbors for each observation.

Serialize the neighbor search results, usually for transmission to other memory spaces.

Parameters
[out]runsOffset to an integer array of length equal to the number of observations.
[out]indicesOffset to an integer array of length equal to the number of neighbors across all observations (i.e., size()).
[out]distancesOffset to a double-precision array of length equal to the number of neighbors across all observations.
Returns
runs is filled with the number of neighbors per observation. indices is filled with the identity of the neighbor indices for each observation, while distances is filled with their distances. Note that these two arrays are filled contiguously for successive observations.
