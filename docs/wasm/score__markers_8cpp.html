Parameters
matAn input log-expression matrix containing features in rows and cells in columns.
groupsOffset to an array of int32_ts with ncells elements, containing the group assignment for each cell. Group IDs should be consecutive and 0-based.
use_blocksWhether or not to compute the statistics within each block.
[in]blocksIf use_blocks = true, offset to an array of int32_ts with ncells elements, containing the block assignment for each cell. Block IDs should be consecutive and 0-based. If use_blocks = false, this value is ignored.
Returns
A ScoreMarkers_Results containing summary statistics from comparisons between groups of cells.
