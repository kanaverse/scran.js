Perform a principal components analysis after blocking on a factor across the cells. This is equivalent to performing a PCA on the residuals after regressing out the factor.

Parameters
matThe input log-expression matrix, with features in rows and cells in columns.
numberNumber of PCs to obtain. Must be less than the smaller dimension of mat.
use_subsetWhether to subset the matrix to features of interest in subset.
[in]subsetOffset to an input array of uint8_ts of length mat.nrow(), indicating which features should be used for the PCA. Only used if use_subset = true.
scaleWhether to standardize rows in mat to unit variance. If true, all rows in mat are assumed to have non-zero variance.
[in]blocksOffset to an array of int32_ts with ncells elements, containing the block assignment for each cell. Block IDs should be consecutive and 0-based.
Returns
A BlockedPCA_Results object is returned containing the PCA results.
